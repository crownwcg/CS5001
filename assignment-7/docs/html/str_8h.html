char* splitString ( const char *  str,
char  ch 
)

Splits a string into a list of strings at the delimiter character ch. It also dynamically allocates storage for a copy of the input string, and replaces each occurence of ch in the copy by the null character ('\0'). It also sets the character after the final null character in the copy to the character '\377' (octal), to mark the end of the list.
